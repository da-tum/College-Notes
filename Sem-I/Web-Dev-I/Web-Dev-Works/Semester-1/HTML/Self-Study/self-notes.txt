
-<a> tag attaching and using one click attribute for attaching difference 
    1. a tag is better for SEo , search engine optimization as it generates and is 
    recognized as a link.
    2. one click attribute does not assign a link in the web but directly routes the image
    to the source link that was added to it at the developing stage of code.
    3.One click attribute has a dependency of JAVAscript , whereas a tag can run without JS 
    being present in the environment.
    4. a tag is recognized as a link by the readers as well, easy for user to navigate.
    easy navigation : link styling , pointer cursor , tab navigation.
